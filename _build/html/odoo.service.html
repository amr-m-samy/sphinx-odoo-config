odoo.service.model.retrying(func, env)[source]

Call func until the function returns without serialisation error. A serialisation error occurs when two requests in independent cursors perform incompatible changes (such as writing different values on a same record). By default, it retries up to 5 times.

Parameters:

  • func (callable) – The function to call, you can pass arguments using functools.partial():.

  • env (odoo.api.Environment) – The environment where the registry and the cursor are taken.
